# ausgRunner

## How to play Demo?

```
- Install CLI tools
$ npm i -g yarn
$ npm install -g @aws-amplify/cli
$ yarn upgrade

- Clone this repository
$ git clone https://github.com/rayleighko/AUSGRunner.git
$ cd AUSGRunner

- Amplify setting
$ amplify configure (Add your AWS Information)
$ amplify init

- Start Apllication
$ yarn
$ yarn start

# Join this Application
# Join this apllication
# Login

# Run to the Goal!
```
